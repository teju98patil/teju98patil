## 🚀 Featured Project

### LLM-Based Classification of European University Names

A data analytics project using a Large Language Model to classify European university names into meaningful categories such as geographic, technical, religious, and named-person references.

The project includes:

- Data cleaning and preparation
- Prompt engineering
- LLM-based multi-label classification
- JSON and CSV output processing
- Manual validation
- Exploratory data analysis
- Data visualization
- Final Quarto report

**Tools used:** Python, Pandas, Anthropic Claude API, JSON, CSV, Quarto, Jupyter Notebook

[View Project](https://github.com/teju98patil/llm-university-name-classification)
